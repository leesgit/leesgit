## AI-Powered Developer | Android 7Y + Fullstack

I build tools that make AI development faster. Published 2 MCP servers on npm, built 28 web/mobile apps with AI.

### What I Build with AI

- **[claude-session-continuity-mcp](https://github.com/leesgit/claude-session-continuity-mcp)** — Zero re-explanation session continuity for Claude Code. 25 tools, knowledge graph, semantic search. `npm install claude-session-continuity-mcp`
- **[mac-pilot-mcp](https://github.com/leesgit/mac-pilot-mcp)** — Self-learning macOS automation. Saves successful sequences as reusable recipes.
- **28 Web/Mobile Apps** — Built with AI-powered 7-mode development system
